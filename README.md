# Einmal ohne, bitte

Nextjs graphcms project.

## Install

`npm install`

## Develop

- Add `.env` file including

```
GQL_CMS_TOKEN=...
GQL_CMS_ID=...
EMAIL=...
MAILGUN_API=...
NEXT_PUBLIC_MATOMO_URL=...
```

- run `npm run dev`

## Graphql Codegen

- run `npm run codegen`

## Tests

- run `npm run test`
